# Run Kilo Code Evals

## Get Started

NOTE: This is MacOS only for now!

Clone the Kilo Code repo:

```sh
git clone https://github.com/Kilo-Org/kilocode.git
cd kilocode
```

Run the setup script:

```sh
cd evals
./scripts/setup.sh
```

Navigate to [localhost:3000](http://localhost:3000/) in your browser.
